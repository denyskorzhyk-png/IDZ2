# Гра в слова

ІДЗ 2: багатокористувацька гра на WebSocket/Socket.IO.

## Суть

Користувач вводить ім'я, створює кімнату або підключається до наявної.
Автор кімнати може почати гру, якщо є мінімум два гравці.

Під час гри сервер контролює:

- черговість ходів;
- щоб нове слово починалось на останню літеру попереднього;
- щоб слово не повторювалось;
- вибування гравця після відмови від ходу;
- визначення переможця, коли лишився один активний гравець.

## Запуск

```bash
npm install
npm start
```

Після запуску відкрити:

```text
http://localhost:3000
```

## Структура

```text
server.js          сервер, Socket.IO, кімнати і правила гри
public/index.html  розмітка екранів
public/app.js      клієнтська логіка і Socket.IO події
public/style.css   стилі
```

## Основні Socket.IO події

Клієнт надсилає:

- `player:name`
- `game:create`
- `game:join`
- `game:start`
- `game:word`
- `game:skip`
- `game:leave`

Сервер надсилає:

- `lobby:update`
- `game:update`
- `game:finished`
- `game:error`
- `game:message`
